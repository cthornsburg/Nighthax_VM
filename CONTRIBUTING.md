## Rules
- **No flags, secrets, credentials, or target IPs**
- No proprietary challenge files or copied paid content
- Prefer official tool links (vendor docs / GitHub)
- Keep notes short and practical

## Suggested PR types
1) Add a tool: edit `resources/tools.md`
2) Add a bookmark: edit `config/browser/bookmarks.md` and (if needed) update `config/browser/bookmarks.html`
3) Improve docs: edit files under `docs/`
